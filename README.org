** Configuration
Following is how environment is configured.
- Environment Variables
  - *Path*:
    - Msys2 - added `${MSYS2 INSTALL DIR}/mingw64/bin/`
    - Git - added `${GIT INSTALL DIR}/cmd/`
    - SBCL - added `${SBCL INSTALL DIR}`
  - *SBCL_HOME*:
    - `${SBCL INSTALL DIR}`
